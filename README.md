# 技能士理论考核模拟系统

在线模拟考试和专项练习系统,支持自定义题库。

## ✨ 功能特性

- 📝 **模拟考试模式** - 45分钟限时,自动评分
- 🎯 **专项练习模式** - 按题型分类练习
- 📒 **错题本功能** - 自动记录错题,支持复习
- 🔊 **语音读题** - 支持题目朗读
- 📱 **移动端适配** - 完美支持手机、平板访问
- ⚡ **即时反馈** - 答题后立即显示对错,自动跳转

## 🚀 使用方法

1. 访问应用
2. 点击"加载自定义题库"上传Excel文件(.xlsx格式)
3. 选择等级和类别
4. 开始模拟考试或专项练习

## 📋 题库格式要求

Excel文件需包含以下列:
- 等级
- 题型(单选题/多选题/判断题/简答题)
- 题目
- 选项A、选项B、选项C、选项D(单选/多选题)
- 答案
- 来源

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript(原生)
- **Excel解析**: SheetJS
- **数据存储**: LocalStorage
- **语音合成**: Web Speech API

## 📱 移动端访问

### 局域网访问
1. 确保手机和电脑在同一WiFi
2. 在手机浏览器输入:`http://[电脑IP]:8080`

### 在线访问
应用已部署到云端,可随时随地访问!

## 🔒 隐私说明

- 所有数据存储在本地浏览器,不上传服务器
- 题库文件仅在客户端解析
- 支持离线使用(PWA)
