# 技能士理论考核模拟系统 v2.3

## 📖 简介

这是一个专为技能士理论考试设计的桌面模拟应用程序,支持自定义 Excel 题库导入、智能组卷、模拟考试和专项练习等功能。

## ✨ 核心功能

### 1. 模拟考试
- **智能组卷系统**:根据等级权重自动抽取题目
  - 例如:二级考试自动包含 10% 一级、70% 二级、20% 三级题目
  - 确保考试难度符合实际考核标准
  - **去重算法**:确保同一张试卷中不会出现重复题目
- **出题规则可视化**:选择等级时,界面实时显示该等级的题目构成比例
- **标准考试流程**:
  - 45 分钟倒计时,时间结束自动交卷
  - 自动判分(80分及格)
  - 错题自动记录到错题本

### 2. 专项练习
- **灵活选择**:支持单选或多选等级进行针对性练习
- **题型分类**:按单选题、多选题、判断题、简答题分类练习
- **即时反馈**:答题后立即显示正确答案和解析
- **进度保存**:自动保存练习进度,下次继续

### 3. 错题本
- 自动收集考试中的错题
- 支持重做错题,巩固薄弱知识点
- 显示错误答案和正确答案对比

### 4. 智能读题 (v2.0 新增)
- **一键朗读**:点击"🔊 读题"按钮,系统自动朗读当前题目
- **自动读题** (v2.1 新增):加载题目或切换题目时,延迟500ms自动朗读,无需手动点击
- **优质女声**:采用 Microsoft Huihui 中文女声,发音自然流畅
- **智能过滤**:自动过滤题目来源等辅助信息,只读题干内容
- **极速响应**:采用独立进程技术,点击即读,支持快速打断和重新朗读

### 5. 人性化设计
- **自动记忆题库**:系统自动记录上次加载的题库路径,下次启动自动加载
- **智能排序**:等级按中文数字(一至六级)自然排序
- **类别筛选**:支持按基站、地宝、窗宝、光学组件等类别筛选题目
- **题目来源追溯**:每道题显示来源 Sheet 和所属等级

## 🚀 快速开始

### 安装要求
- Windows 7/10/11 操作系统
- 无需安装 Python,直接运行 `main.exe`

### 使用步骤

#### 1. 首次使用
1. 双击运行 `main.exe`
2. 点击"加载自定义题库"按钮
3. 选择符合格式要求的 Excel 题库文件(`.xlsx`)
4. 系统会自动记忆该文件,下次无需重新加载

#### 2. 开始模拟考试
1. 在"请选择等级"下拉框中选择考试等级(如"二级")
2. 查看下方显示的出题规则(如:10% 一级、70% 二级、20% 三级题目)
3. (可选)在"请选择类别"中筛选特定类别
4. 点击"开始模拟考试"
5. 答题过程中:
   - 左侧导航栏显示所有题目,绿色表示答对,红色表示答错
   - 右上角显示剩余时间
   - 单选题和判断题选择后自动提交
   - 多选题需点击"确认答案"
6. 完成后点击"交卷"查看成绩

#### 3. 专项练习
1. 点击主界面"专项练习模式"
2. 勾选一个或多个等级(支持多选)
3. 点击"刷新题目数量"查看各题型的题目数
4. 点击对应题型按钮(如"单选题(120题)")开始练习
5. 练习模式下,答错的题目会标记为红色,答对后才变绿色
6. 点击"退出练习"返回主页

#### 4. 查看错题本
1. 点击主界面"我的错题本"
2. 浏览历史错题
3. 重新作答,巩固知识点

## 📋 题库格式要求

Excel 文件(`.xlsx`)需包含以下列:

| 列名 | 说明 | 示例 |
|------|------|------|
| 考题类型 | 题目类型 | 单选题/多选题/判断题/简答题 |
| 题目 | 题目内容 | 以下哪项是正确的? |
| 选项A | 选项A内容 | 选项内容A |
| 选项B | 选项B内容 | 选项内容B |
| 选项C | 选项C内容(可选) | 选项内容C |
| 选项D | 选项D内容(可选) | 选项内容D |
| 答案 | 正确答案 | A 或 AB 或 √/× |
| 等级 | 题目等级 | 一级/二级/三级等 |
| 来源 | 题目来源(可选) | 基站/地宝/窗宝等 |

**注意事项**:
- 判断题的答案可以是 `√`/`×` 或 `A`/`B`
- 多选题答案需按字母顺序排列,如 `ABC`
- 支持多个 Sheet,加载时可选择需要的 Sheet
- 每个 Sheet 名称会作为题目来源显示

## 📊 评分规则

| 题型 | 每题分值 | 数量 | 小计 |
|------|----------|------|------|
| 单选题 | 1分 | 40题 | 40分 |
| 多选题 | 2分 | 10题 | 20分 |
| 判断题 | 1分 | 10题 | 10分 |
| 简答题 | 10分 | 3题 | 30分 |
| **总计** | - | **63题** | **100分** |

**及格线**: 80分

## ❓ 常见问题

### Q: 无法加载题库怎么办?
**A**: 请检查:
1. Excel 文件格式是否为 `.xlsx`
2. 是否包含必需的列(考题类型、题目、答案等)
3. 文件是否被其他程序占用

### Q: 为什么某些等级无法开始考试?
**A**: 该等级题目数量不足。每个等级至少需要:
- 单选题 40 道
- 多选题 10 道
- 判断题 10 道
- 简答题 3 道

### Q: 点击"读题"没有声音怎么办?
**A**: 请检查:
1. 电脑音量是否开启
2. 系统是否安装中文语音包(Windows 通常自带)
3. 如果仍无声音,请尝试重启程序

### Q: 为什么试卷中会出现重复题目?
**A**: v2.0 已修复此问题。如仍出现,请确保使用最新版本。

### Q: 如何更换题库?
**A**: 点击主界面"加载自定义题库"按钮,选择新的 Excel 文件即可。

### Q: 练习进度保存在哪里?
**A**: 保存在程序目录下的 `practice_progress.json` 文件中。

### Q: 错题本数据保存在哪里?
**A**: 保存在程序目录下的 `error_questions.json` 文件中。
