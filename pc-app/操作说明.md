# 技能士考试刷题系统 V3.1 操作说明

## 1. 启动软件

1. 将 `SkillExam-V3.1-win-x64-portable.zip` 完整解压到任意可写目录。
2. 保持 `技能士考试刷题系统.exe` 与 `exam_bank` 文件夹的相对位置不变。
3. 双击 `技能士考试刷题系统.exe` 启动，不需要安装软件或 .NET 运行时。

首次启动会自动读取随程序附带的 `exam_bank\PE 技能士题库.xlsx`。也可以在主页选择其他 Excel 题库；加载成功后，软件会记住最近使用的题库和工作表。

## 2. 开始答题

- **模拟考试**：在主页选择等级和题目类别后开始考试，答题页显示倒计时，完成后点击“交卷”。
- **专项练习**：选择等级、类别和题型后开始练习，进度会自动保存。
- **错题本**：查看历史错题、重新作答，或清空错题记录。
- **上一题/下一题**：可随时手动切换题目；左侧题号也可直接定位。

单选题和判断题点击选项后立即判定；多选题选择完成后点击“确认答案”；简答题输入答案后提交或查看参考答案。

## 3. 自动跳到下一题

回答正确且当前不是最后一题时，软件会在设定的延迟后自动进入下一题。默认延迟为 **1.0 秒**。回答错误时不会自动跳题，便于核对正确答案；等待期间手动切题会取消当前自动跳题任务。

调整延迟：

1. 在主页点击“设置”。
2. 在“答题与显示”中拖动“跳题延迟”滑块。
3. 可选范围为 **0–3.0 秒**，每次调整 **0.1 秒**。设为 0 秒时，答对后立即跳题。
4. 点击“保存设置”。新设置会用于之后开始的答题会话。

点击“恢复默认设置”会将跳题延迟恢复为 1.0 秒，但不会删除题库路径、练习进度或错题数据。

## 4. 语音朗读

设置页可开启或关闭自动读题，并选择系统语音和语速。答题页的“读题”按钮可以手动朗读当前题目。若没有中文语音，请在 Windows 语言与语音设置中安装中文语音包。

## 5. 用户数据与备份

用户数据库、日志和备份位于：

```text
%LOCALAPPDATA%\PEGroup\SkillExam
```

在设置页点击“查看数据目录”可直接打开该目录，点击“备份数据库”可生成当前数据库备份。便携版程序目录可以移动或替换，用户数据不会因此丢失。

## 6. 更新软件或题库

- **更新软件**：关闭旧版，解压新版便携包后运行新的 EXE。用户数据保存在系统数据目录，会继续沿用。
- **更新默认题库**：关闭软件后替换 `exam_bank\PE 技能士题库.xlsx`，保持文件名和表头结构不变。
- **使用自定义题库**：在主页重新选择 Excel 文件，并按提示选择工作表。

## 7. 常见问题

**软件无法启动**：确认压缩包已完整解压，EXE 与 `exam_bank` 文件夹没有被单独移动，并检查 Windows 安全中心是否拦截文件。

**点击读题没有声音**：检查系统音量、设置页的语音选择及 Windows 中文语音包。

**答对后没有自动跳题**：最后一题不会自动跳转；若在等待期间手动切题，原自动跳题任务会取消。可回到设置页确认跳题延迟并保存。

**如何保留数据**：正常更新便携版不需要复制数据库。需要额外留档时，在设置页使用“备份数据库”。
