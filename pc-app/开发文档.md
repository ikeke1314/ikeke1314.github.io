## 🔧 核心功能实现

### 1. 智能组卷算法

```python
def start_exam(self, level, selected_sources=None):
    # 获取权重配置
    weights = self.LEVEL_WEIGHTS.get(level, {level: 1.0})
    
    # 按权重从各等级抽取题目
    for source_level, weight in weights.items():
        count = int(required_count * weight)
        questions = self.bank.get_questions_by_level(source_level)
        # 筛选来源
        if selected_sources:
            questions = filter_by_source(questions, selected_sources)
        # 随机抽取
        selected = random.sample(questions, count)
```

### 4. 等级排序

```python
def sort_levels(self, levels):
    chinese_nums = {'一': 1, '二': 2, '三': 3, '四': 4, '五': 5, '六': 6}
    
    def get_level_value(lvl_str):
        for char in lvl_str:
            if char in chinese_nums:
                return chinese_nums[char]
        return 999
    
    return sorted(levels, key=get_level_value)
```

## 📊 数据流程

### 考试流程
```
用户选择等级 → ExamManager.start_exam()
    ↓
根据权重从题库抽取题目
    ↓
MainWindow 显示题目
    ↓
用户答题 → ExamManager.submit_answer()
    ↓
判断正误,记录答案
    ↓
用户交卷 → ExamManager.calculate_result()
    ↓
计算总分,保存错题
    ↓
显示成绩
```

### 练习流程
```
用户选择等级和题型 → QuestionBank.get_all_questions()
    ↓
获取所有符合条件的题目
    ↓
加载上次进度(如果存在)
    ↓
用户答题 → 即时反馈
    ↓
保存进度(题目索引 + 错题列表)
```
