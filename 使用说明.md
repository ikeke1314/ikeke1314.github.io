# 技能士理论考核模拟系统 - 使用说明

## 📱 系统概述

本系统是一个基于Web的在线考试练习平台,**完美支持手机、平板和电脑**,随时随地刷题学习。

### 核心功能

- 🎯 **模拟考试** - 按等级智能组卷,45分钟限时考试
- 📚 **专项练习** - 自选等级和题型,针对性练习  
- 📒 **错题本** - 自动记录错题,支持重复练习
- 🔊 **语音读题** - 点击"读题"按钮,听题答题
- 💾 **本地存储** - 数据存储在浏览器,隐私安全

## 🚀 快速上手

### 第一步:准备题库

1. **题库格式要求**

Excel文件(.xlsx或.xls)需包含以下列:

| 必填列 | 说明 | 示例 |
|--------|------|------|
| 考题类型 | 题型 | 单选题/多选题/判断题/简答题 |
| 题目 | 题目内容 | 以下哪项是... |
| 答案 | 正确答案 | A 或 AB 或 √ |
| 一级~六级 | 等级标记 | 任意非空值(如√、是、1) |

| 选填列 | 说明 |
|--------|------|
| 选项A-E | 选项内容(判断题为√和×) |
| 来源 | 题目分类/来源 |

2. **题库示例**

```
考题类型 | 题目           | 选项A | 选项B | 答案 | 一级 | 二级
--------|---------------|-------|-------|------|------|-----
单选题   | 正确的是?     | 选项1 | 选项2 | A    | ✓    |
多选题   | 以下正确的有? | 选项A | 选项B | AB   |      | ✓
判断题   | 某说法正确吗  | √     | ×     | √    | ✓    | ✓
```

### 第二步:加载题库

1. 打开应用(网页或本地HTML文件)
2. 点击**"📂 加载自定义题库"**按钮
3. 选择您准备好的Excel文件
4. 勾选要加载的工作表(支持多选)
5. 点击**"确定加载"**

✅ 加载成功后会显示题目总数

### 第三步:开始练习

#### 模式一:模拟考试

适合全真模拟,检验学习成果

1. **选择等级** - 从下拉框选择目标等级(如:三级)
2. **选择类别**(可选) - 筛选特定分类题目
3. **查看规则** - 系统自动显示出题比例
4. **开始考试** - 点击"开始模拟考试"

**考试规则:**
- ⏱️ 时间:45分钟倒计时
- 📊 题量:单选40题、多选10题、判断10题、简答3题
- 🎯 计分:单选/判断1分,多选2分,简答10分,满分100分
- ✅ 及格:80分

**答题技巧:**
- 答对自动跳转下一题(0.5秒延迟)
- 答错停留当前题,可查看解析
- 支持上一题/下一题导航
- 简答题需点击"查看答案"
- 多选题需点击"确认答案"

#### 模式二:专项练习

适合针对性强化训练

1. **选择等级** - 可多选,如同时练习二级+三级
2. **选择类别**(可选) - 筛选特定分类
3. **选择题型** - 点击题型卡片(显示题目数量)
4. **开始练习** - 无时间限制,可随时退出

**练习特点:**
- 🔄 支持中断和恢复
- 📈 实时显示可用题目数
- 🎯 专注某一题型

### 第四步:查看错题

#### 考试结束后

1. 系统自动显示成绩和错题数
2. 点击**"查看错题"**按钮
3. 查看错题详情:题目、您的答案、正确答案

#### 随时查看错题本

1. 在首页点击**"📒 我的错题本"**
2. 浏览所有历史错题
3. 支持**"清空"**错题本

## 💡 常见问题

### Q1: 题库导入失败怎么办?

**A:** 检查以下几点:
1. 文件格式是否为`.xlsx`或`.xls`
2. 必填列是否齐全(考题类型、题目、答案、至少一个等级)
3. 表格是否从第4行开始(前3行可以是标题)
4. 答案格式是否正确(单选:A,多选:AB,判断:√或×)

### Q2: 为什么有些题目没有加载?

**A:** 可能原因:
- 该题缺少必填信息
- 没有勾选对应的Sheet
- 等级列全为空

打开浏览器控制台(F12)查看详细日志

### Q3: 错题本为什么是空的?

**A:** 
- 如果全部答对,错题本为空是正常的
- 刷新页面后检查浏览器控制台是否有错误
- 确保浏览器允许LocalStorage存储

### Q4: 可以在手机上使用吗?

**A:** 完全可以!
1. **iOS**: 用Safari打开HTML文件或在线网址
2. **Android**: 用Chrome打开
3. 建议添加到主屏幕,像App一样使用

### Q5: 数据会丢失吗?

**A:** 
- 错题和进度存储在**浏览器本地**
- 只要不清除浏览器数据就不会丢失
- **注意**:隐私/无痕模式下退出会清空数据
- **注意**:不同浏览器数据不共享

### Q6: 如何清空错题本?

**A:** 
1. 进入错题本界面
2. 点击右上角**"清空"**按钮
3. 确认后立即清空

或在控制台执行:
```javascript
localStorage.removeItem('skill_exam_error_book');
```

### Q7: 语音读题没有声音?

**A:**
- 确保设备音量已打开
- 某些浏览器需要用户交互后才能播放声音
- iOS Safari可能需要手动允许自动播放

## 🔧 高级技巧

### 查看存储数据

打开浏览器控制台(F12),输入:

```javascript
// 查看错题本
console.table(JSON.parse(localStorage.getItem('skill_exam_error_book')));

// 查看练习进度  
console.log(JSON.parse(localStorage.getItem('skill_exam_practice_progress')));

// 查看设置
console.log(JSON.parse(localStorage.getItem('skill_exam_settings')));
```

### 导出错题数据

```javascript
// 导出错题为JSON
const errors = localStorage.getItem('skill_exam_error_book');
const blob = new Blob([errors], {type: 'application/json'});
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = '错题本.json';
a.click();
```

### 批量导入错题

```javascript
// 从文件读取JSON并导入
const errors = [...]; // 您的错题数据
localStorage.setItem('skill_exam_error_book', JSON.stringify(errors));
```
